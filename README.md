# e-commerce

## Description

This is a backend application for an ecommerce website. The app uses Sequlize to create and store data in tables for product items. 
The app have categories, products, and tags tables for the products. 

## Technologies

-  Node.js
-  Express.js
-  Javascript
-  Sequelize
